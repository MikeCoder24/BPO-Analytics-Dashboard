Call Center Analytics Dashboard
📶 Overview

This project presents an end-to-end analytics solution for a call center environment using Power BI.

It tracks key KPIs such as SLA, Average Handle Time (AHT), CSAT, and Resolution Rate to evaluate operational efficiency.

📊 Key Features
SLA monitoring based on handle time thresholds
Agent performance analysis
Call volume trend analysis
Queue distribution insights
KPI indicators with dynamic color alerts
🧠 Key Insights
SLA is below target (51%), indicating operational inefficiencies
Call volume decreases after day 10
High-performing agents show variability in efficiency
🛠 Tools Used
Power BI (DAX, Data Modeling)
Python (Pandas, NumPy)
SQL
